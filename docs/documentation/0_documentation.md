# Documentation

Welcome to the Telekit documentation — the official source of knowledge from the developer.
Here you’ll find everything you need to understand how the library works and how to use its features.

If any questions arise, feel free to join the [Telekit community](https://t.me/+wu-dFrOBFIwyNzc0), where I and other members can assist you.

If you encounter a bug or unexpected behavior, please report it in the GitHub Issues section.
